Spanish Bible API API Reference

An API to query spanish (Reina Valera 1960) bible verses and books.

API Endpoint
http://bibledemoapi-env.eba-ej2dwwya.us-west-2.elasticbeanstalk.com/api
Contact: alphaelena@gmail.com
Schemes: http
Version: 2.0

Paths

GET /books

Return a list of bible books

200 OK
Books

Success response

Response Content-Types: application/json
Response Example (200 OK) 
[
  {
    "id": "string",
    "testament": "string",
    "name": "string",
    "chapters": [
      {
        "id": "string",
        "osis_end": "string",

      }
    ]
  }
]

GET /books/{bookId}/verses/{range}

Return a list of verses of a given bookId and a range in the format: startChapterNumber:startVerseNumber-endChapterNumber:endVerseNumber

bookId: string
in path

(no description)

range: string
in path

Example ranges: [1, 1:10, 1:10-2, 1:10-2:5, 1-3]

200 OK
Verses

Success response

Response Content-Types: application/json
Response Example (200 OK) 
[
  {
    "id": "string",
    "cleanText": "string",
    "reference": "string",
    "text": "string",
    "chapterId": "string"
  }
]
